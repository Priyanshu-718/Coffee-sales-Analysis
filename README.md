Coffee Sales Dashboard

Author: Priyanshu K Choubey

Project Overview
This project is a Power BI dashboard built using the Coffee Sales Dataset. The objective of this project is to analyze daily, monthly, and yearly sales performance for a coffee shop, and to understand customer behavior across different times of the day, payment methods, and coffee types.
The dashboard presents a comprehensive view of sales trends, popular products, and performance insights that can help business owners make data-driven decisions.

Dataset Used
Dataset: Coffee Sales Dataset from Kaggle
The dataset includes fields such as coffee name, money, hour of day, date, payment types, weekday, and more.
Key Insights Displayed
The dashboard highlights the following:
Total revenue generated
Total number of orders in 2024
Morning sales
Afternoon sales
Night sales
Sales by specific products such as latte
Card payment count
Average
morning, afternoon, and night sales
Sum of money by coffee type
Yearly sales comparison
Latte sales by weekday

A detailed table showing total sales by coffee type and hour of day
Day-based filtering using a weekday slicer

Dashboard Features
Interactive slicers for weekday selection
KPI cards for important business metrics
Bar charts showing product category performance
Line charts for year-wise comparison
Donut chart for weekday-wise latte sales
A detailed tabular view for deeper analysis
Organized layout with branded sections such as sales analysis, technology, and types of coffee provided



Conclusion

This project demonstrates the use of Power BI for analyzing retail coffee sales and understanding customer purchasing patterns. It also showcases the ability to transform raw data into clear visual insights useful for decision-making.
